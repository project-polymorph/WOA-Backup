#   厚土堆肥





🧑🏻‍🌾 ：先混合多种有机生物体，同时在土壤表层掺入腐熟的牛粪土，让其在土壤深处逐渐发酵和形成稳定且丰富活跃的微生物群落。自下而上皆厚土。

●  种植土表层5cm直接用一些腐熟的小牛粪土，作物的种子和根茎能够直接吸收其中的营养物质。理所当然的，第一茬作物会长得很壮，但第二茬就不好说了，肥力到那个时候可能变得一穷二白。
●  在土壤底部25-40cm处，埋进去果皮蔬菜厨余，腐烂的木头和落叶，牛马的粪便等等，让它们逐步发酵释放养分，最终变成土壤的一部分。在土壤中的微生物在分解与被分解的过程里，生态系统会不断自我壮大并趋于稳定。在这样的土壤环境里生长的作物，会一茬比一茬的壮。











***农场的准备***

农场的同事开来了运输用的三轮车，并为六名志愿者准备了手套、草帽、铁锹、等工具和一个花坛。志愿者们利用提供的编织袋收集好了已经腐熟完成的小牛粪，这些是制作堆肥的“秘密武器”。








***一起堆肥填土***

在小花坛的底部长着杂草，我们决定尝试通过最少的扰动土壤的方式，不进行除草与翻根，尝试直接在土表堆肥。我们将准备好的各种堆肥材料一层层填入花坛中。




* **a.  白色大桶：我们将使用白色的大桶中，未完全发酵的青草肥倒入地里。
* **b.  腐木和干草：腐木里含有大量的微生物和真菌，干草能够中和，可以加速后面的发酵堆肥。
* **c.   菜籽饼：磨碎的菜籽饼，可以为微生物提供氮源。
* **d.   十厘米土：添加薄土，组成一个“三明治”。
* **e.   果皮等厨余：厨余是志愿者们贡献的重要有机物来源。
* **f.   十厘米土（注意隔开酸碱）：再次添加薄土，以覆盖厨余，减少气味的散发。
* **g.  一袋草木灰：我们接着添加草木灰，这需要一定的经验和对具体土壤的判断，以确保酸碱平衡，提供有益于微生物的矿物质。
* **h.  十厘米土：再次添加薄土，确保酸碱平衡的同时提供更多的有机物质和微生物。
* **i.  小牛粪：这一步为堆肥增加了一定的容错率，我们将已经发酵好的小牛粪添加到堆肥堆中，小牛粪已经可以直接为植物提供营养。
* **j.  十厘米土：最后一次添加薄土，将堆肥物料完全覆盖。



微生物将会慢慢发酵，在升温的过程中，杀死那些强壮的具有毒性的细菌，随着时间的推移，这些材料就变的松软无味，形成适宜植物生长的环境。
在家庭中我们也可以以建造一个小生态圈的方式，尝试进行这样的堆肥，最后种出自己的共生菜盆。用纸箱代替腐木和树叶，用家庭中的蔬果厨余来提供有机质，用鸡蛋壳、咖啡渣增加钙和氮等，提供丰富的基质环境。
