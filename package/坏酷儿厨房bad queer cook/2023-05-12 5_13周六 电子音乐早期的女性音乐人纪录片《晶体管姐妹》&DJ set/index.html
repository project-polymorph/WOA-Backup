简介：
想想早期的电子音乐，你可能会看到男人在摁按钮、拧旋钮和推推子。虽然电子音乐通常被认为是男孩俱乐部，但事实是，从一开始，女性就参与了发明设备、技术和理念的工作，这些设备、技术和理念将在未来几年定义声音的形态。作为电影的主题之一，劳里·斯皮格尔解释说：“当女性作曲的可能性本身就存在争议时，我们女性尤其喜欢电子音乐。电子产品让我们制作出可以被其他人听到的音乐，而不必面对男性主导机构的严肃对待。”
《晶体管姐妹》是一部 2020 年的纪录片，由丽莎·罗夫纳 (Lisa Rovner)执导，是她的导演首作。这部纪录片由罕见的资料和档案片段组成。它记录了电子音乐中作为先驱但鲜为人知的女性的故事。多亏了电子设备，这些作曲家发现了男性主导的传统音乐世界所不允许的自由和创造力空间。这部电影将我们带入了她们工作室实验室的中心，并进入五颜六色的电缆、磁带碎片、早期计算机和巨大合成器的纠缠中。

本片由劳里·安德森 (Laurie Anderson)讲述，片中的十位艺术家是Maryanne Amacher、Bebe Barron、Suzanne Ciani、Delia Derbyshire、Pauline Oliveros、Daphne Oram、Éliane Radigue、Clara Rockmore、Wendy Carlos和Laurie Spiegel。
