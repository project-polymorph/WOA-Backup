不想写太多的 intruding，但我相信很多人对于 CNKI 上读文献是有需求的。总之就是没事多读书多读学术成果，肯定会对自己的知识体系构建有帮助。


这篇指南针对的是北师珠的同学或者校友，其他学校可能不适用，但大方法其实还是相同的。而用 iPad 的起因就是，背电脑确实是一件很麻烦的事情，何况 Apple Pencil 用来做笔记比电脑方便多了。


教程有两种版本，一个是文字版，一个是视频版。建议先看文字版，然后不明白的再看视频版。


首先，在 iPad 上安装这两个 app：

  • EasyConnect；

  • CAJ 云阅读。




然后在 iPad 的浏览器打开这个页面：http://library.bnuz.edu.cn/fw/xwfw.htm，仔细阅读《图书馆校外访问功能开通》这篇说明，以及下面的两个 Word 文档。



虽然学校说是为教师和毕业生开放，但在校生似乎也是可以使用的。


第一步：打开 EasyConnect，输入图片中的地址，并点击连接。

第二步：输入你的帐号和密码，并勾选「记住密码」和「自动登录」，然后点击「登录」。如果你不知道账号密码，在上面的《图书馆校外访问功能开通》里有告诉你。

第三步：点击 allow，然后验证指纹 / 密码 / 你的大饼脸。

第四步：连接成功，在「常用数据库资源」里，第一个就是 CNKI，点击它。

第五步：搜索你想要的文献内容，然后点击下载（黄色的小箭头）。

第六步：点击「在“CAJ 云阅读”中打开」。

第七步：徜徉在知识的海洋里吧。


视频教程请在微信公众号后台回复「CNKI」获取。


别的学校使用方法类似，但需要先查看学校是否有购买知网文献，对什么人员开放，以及对应的 VPN 客户端。


除了 CNKI 之外，其它数据库资源如超星视频、超星图书、方正图书等等，只要是学下购买了并共享了之后，这个方法都可以使用。



The END.
