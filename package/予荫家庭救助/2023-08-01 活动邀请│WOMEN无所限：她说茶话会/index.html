{

割礼

}


2009年，一部由自传改编的电影《沙漠之花》被搬上荧幕并在三十多个国家热映。这部电影的原型，是曾当选福布斯30位全球女性典范之一，被誉为“非洲女权斗士”的国际超模——华莉丝·迪里。

这部电影在在向观众讲述华莉丝与地狱抗争的成长之路时，也将一种残忍血腥的习俗在全世界揭露——割礼。

女性割礼的意思是，割掉女性部分或全部的外生殖器，再进行缝合，以保证她成为一名“忠贞的女子”。由于实行女性割礼的地区大多极度贫穷落后，很多人并没有条件到医院进行手术，而是在没有麻醉的情况下，使用小刀、铁片甚至石片进行切割，然后再用针线或者荆棘进行缝合，这直接导致了大量女性感染疾病或大出血而死亡。


割礼示意图（图源自网络）


正在进行的割礼仪式（图源自网络）


而在非洲大陆，还存在着许多迫害女性的陋习，例如埃塞俄比亚的盘唇习俗，肯尼亚马赛人的一夫多妻制，都昭示着落后的社会形态和尚未开化带给女性的痛苦。

压迫也不只存在于非洲，在过去的大部分时间里，世界上各个角落的女性都在承受着时代施加在她们身上的“割礼”。在中国近千年的缠足历史中，人们歌咏盛赞骨折且畸形的小脚，冰冷华丽的辞藻下，是被中国女性的血与痛浸染的一张张裹脚布;而在西欧的猎巫运动，人们更将巫师信仰这一把利刃，刺向了无数被诬陷迫害致死的妇女。

割礼或裹小脚，在将痛苦施加在女性肉体上时，也在用贞洁和封建礼教将她们洗脑。而这种思想上的“割礼”，也存在于开化进步的现代社会，随着生产力的高速发展，许多饱含歧视的观点和不平等的待遇也被施加在了女性身上。这是时代发展给女性带了的新的思想上的“割礼”和困境。

华莉丝用不屈的抗争改写了命运为她编织的悲剧，这鼓励了无数身陷绝境的女性。而我们也希望，现代的女性用自己的声音和力量，与困境反抗，绽放出属于自己的生命之花。

{

活动介绍

}



每年的7月30日为“非洲妇女日”，旨在保证妇女的平等权利不受侵犯，促进女性得到保护与人权尊重。Savus一直致力于保护妇女权益和推动性别平等，值此2023年非洲妇女日来临之际，特举办本次“WOMEN无所限：她说茶话会”活动，关注女性议题，推进性别平等。

本次茶话会邀请到学者与我们一起在对割礼问题和受害女性成长问题的思考下，就非洲女性遭遇的困境进行交流和探讨，促进女性议题的思想交流。
