这是一个觉醒的故事

这是通过自己的方式

来选择自己想要的人生

生命终结

对于“我”来说

不是结束

而是一场新的开始

“我”希望“我”的人生

由自己定义

活出自己真正的样子

其实

“我”同你一样

焦虑和孤独

只是

现实生活已经把“我”浸润得麻木

“我”无力抗拒

只能任由现实社会撕扯

向来只属于自己

该如何选择

只希望你遵循自己的内心







罗曼·罗兰曾言：“世上只有一种英雄主义，就是在认清生活真相之后依然热爱生活。”亲密关系、原生家庭、自我认同、生活意义……普利策奖经典戏剧《晚安，妈妈》将观众拽入了对这一系列心理学、社会学问题的追问与思考之中，而这些无不是这个时代绕不过去的焦点话题。

值此母亲节来临之际，Savus推出母亲节剧本演读活动，将在母亲节当天呈现一场声音表演。我们将戏剧空间搬到线上，实现舞台的无边界化，共同成就一个人人可参演、人人可观演的实验性体验。剧本演读，即剧本的台词表演，在无画面呈现的前提下仅通过台词演绎来渲染情感，推动剧情。

我们鼓励素人参与到这场演出中，现对剧目《晚安，妈妈》进行剧组征募如下：


— save love, save us —





【岗位征募】


导演（1位）


1.戏剧影视导演、编剧等专业学生或从业者，具备导演能力和经验；

2.优先选择对《晚安，妈妈》剧本有过了解和学习经验者；

3.能花时间和精力指导排练；

4.认真负责，有耐心，乐于交流和沟通，能够服从组织安排。



演员（2位）


1.具备一定台词功底即可，不要求有戏剧表演或播音主持专业背景；

2.声音性别为女性；

3.能花时间和精力进行排练；

4.有团队精神，善于沟通，责任心强能够服从组织安排。


— save love, save us —







【试镜内容】


导演


a.准备三分钟自我介绍，可加入声音类的才艺展示；

b.对晚安妈妈剧本的个人理解进行自由阐述；

c.对晚安妈妈剧本演读会编排的初步想法。


演员


a.准备三分钟自我介绍，可加入声音类的才艺展示；

b.准备一段文字素材，题材和内容不限，进行现场台词演绎；

c.对给定晚安妈妈剧本选段进行现场演绎。


— save love, save us —
