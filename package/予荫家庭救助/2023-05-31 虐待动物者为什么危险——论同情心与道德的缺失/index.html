虐待动物者为什么危险——论同情心与道德的缺失

虐杀动物者为什么危险

——论同情心与道德的缺失


近期在网上有一则社会事件，山东一男子李浩酒后夜里将家养柯基狂砍致死，而这甚至还算不上是“新闻”。

在这个事件里，该男子将结婚伴侣所养、甚至也曾为自己爱宠的柯基犬以残忍的方式结束生命，现场的血腥图片想必胜过千言万语。而除了最直观的视觉冲击，虐杀动物还反映了什么其他问题？我们可以从以下几个方面进行探讨。

1. 同情心的缺失反映出来人格的残缺

2. 对幼小生命的漠视-对其他生命的忽视与不尊重

3. 家人同样漠视生命的态度

4. 虐待动物是反社会人格的基本表现之一



1、同情心的缺失——人格的残缺


在此我不再论述家养宠物与人类的关系，作为爱犬它不曾为“威胁到自己生命”的生物，从体格上，男主人也形成了绝对性的压制。但在自己生命未受到威胁，却将其视为出于寄托满足自身弑杀欲望的生物体、并将其残忍杀害时，这无疑是一种高于“冷漠”一词的冷血，也很直观地反映出这种人缺失的同情心。



2、对幼小生命的漠视

——对其他生命的忽视与不尊重  


同情心的缺失是无法推己及人、无法感受作为人应有的种种心理的原因之一。他们极端自负，没有普世的善恶标准，所作所为皆服从于自己的欲望。而当其面对从生物视角的“弱者”时，他们本质也是将那些弱于自己的动植物、无机体如这只柯基犬一般对待。更进一步说，是弱势群体，甚至这个“弱势”也许仅仅是地位、力量弱于自己的所有人。

该男子的妻子遭受到了家暴，也是他对其他生命并无尊重之心的直观表现，甚至是在随意践踏着生命。既然能对爱犬如此，对待弱于自己的人时，是否也会如杀害宠物般肆意处置他人生命？



3、家人同样漠视生命的态度


在其妻子的论述中，男主人在杀害自己爱犬后非但不觉得有任何过错，他的家人也同样是认为此事是小题大做，不难看出冷漠的人情土壤里更容易结出罪恶的果。



4、虐待动物是反社会人格的基本表现之一


反社会人格人群的基本共同点里有一条：虐待动物。虽说虐待动物者就是反社会人格这一推导过程尚不能成立，但虐待动物本身就是一种强者对于弱者生命权肆意践踏的表现，而且在反社会人格里确实成为了极大的共同点。

反社会人格者对于人类社会的危害令人发指。他们漠视规则、生杀予夺，其实与同情心的缺乏同源——道德刹车机制。那条康德提到道德的名言里将道德与星空所放在一起，道德的存在会让我们拥有求真、向善、憧憬美好的方向。而道德的缺失必定对于现今人类已有的普世理念而言是残忍的。道德刹车机制则是能够在被某种欲望绑架时与自己谈论来龙去脉，能够静下来思及自身所作所为是否符合道德。如若缺乏这样的机制，那么将阻止千钧一发间会酿成大错的可能斩断，人也成为被欲望支配的生物。



在这样的事件里，更让人毛骨悚然的并非一只动物的死亡，而是这座冰山在海平面底下庞大的、盘根错节的种种。这并非一件个例事件，它存在于过去与现在。
