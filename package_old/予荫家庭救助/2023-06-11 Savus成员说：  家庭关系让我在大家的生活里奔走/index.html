Savus成员说： 家庭关系让我在大家的生活里奔走

原创 东亚式家庭救助 予荫家庭救助
2023年06月11日 09:09 北京


Savus成员说：

家庭关系让我在大家的生活里奔走



近期一直在想关于我自己，还有我身边朋友的家庭关系。


//

我一直认为我是一个很容易依赖别人的人。有一次和朋友提起，我很容易对老师产生一种依赖性。简单来说就是喜欢和老师接触，和老師打好關係。她百思不得其解，因为她遇到老师都恨不得离得远远的，生怕老师找上她。而我卻與她相反。我認為一個老師對於孩子的影響是極大的，老師當天的心情，懷著什麼樣的心態上課，對於學生的影響都非常的大。有句話是這麼說的，孩子是看著父母的背影成長的，也同樣的可以放在學生與教師的關係上。


我相信有許多人無法認同我的觀點，因為這樣子無形之中給教師這群體增加了一種壓力。我明白這樣子的想法會對老師造成一定的困擾，並且這也對我自己產生了一定的心理壓力。那我這種想法究竟是怎麼造成的呢？解鈴還須繫鈴人，我決定從能接觸到的，也算比較熟悉的家庭關係開始入手。


我问了自己很久，终于有一天，给了自己一个答案。

//

简单说下我的家庭情况吧，我妈妈在国企工作，每天忙得不可开交，每年能正常下班的日子两只手可以数的过来。而我爸爸，从我记事开始就在外省工作了，每年能见面的日子少之又少。


幼兒園的時候，妈妈来接我放学是十分难得的事，放学了大家都高高兴兴地和家长回家，而我在学校等着，等两三个小时，直到天都黑了，才见到匆匆赶来的妈妈。上了小学，我被送进托管，与家里人交流的时间又变得更短更短了。


失望不是一瞬间养成的。从小开始学跳舞，也避免不了有演出，但是大部分的演出，我的妈妈基本都没来看。她把我的一切，都選擇交給了老師。


可能上面说的事都很小很小很小。但或许因为我是双鱼座，或者性格天生就是这样，这些小事积少成多，在我心中也成了一个心结。这也间接导致我和老师的接触比较多，我会对老师产生依赖，会选择去讨好老师，会因为一句随口的表扬开心。




//

在 Savus 做了对接工作之後，发现东亚式家庭里的很多負面影響就是源自家庭陪伴的缺少。我家可能不算典型的东亚式家庭吧，也不能当典例参考。父母有爱不会表达，家庭间缺少交流，缺少陪伴，孩子在这样没有安全感的环境下成长，一般会变得心思细腻、敏感，形成讨好型人格。


采访了一位在知乎上认识的好朋友，她比我小一岁，这时候正在经历中考的磨练。她是爷爷奶奶带大的，虽然说身边有很多爱她的亲人，但唯独缺少了母亲的陪伴。听她奶奶说，她的母亲在很小的时候就不要她了，因为她是女孩子。为了不要她，在孩子刚断奶的时候就抛夫弃子回娘家。但令人不解的是，她母亲在她小学五年级的时候回来了，但却又因为一件极小极小的事离开了——因为她想吃零食，和母亲吵了一架。


我一直认为人与人之间是有一定的磨合期的，就好像谈恋爱，总是会出现分歧，但往往能走到最后的，都是越吵越恩爱的两个人。同样的也能放在亲子关系身上，只是亲子关系不能做选择，也不能随意离开罢了。一起生活半辈子的母女都会吵架，更何况从小没在一起生活的血缘母亲呢？


而她父亲那边也再娶了，和她的继母生了个妹妹。她本就不多的爱，更少了。小时候成天和妹妹打架，现在长大，也就还好了。再后来，父亲离异，女方带着妹妹在城市生活。她生活的地方在城乡结合部，可能是年纪小，她觉得妹妹有点趾高气昂，妹妹一回来姐妹还是会斗嘴。


这样的家庭，造就了现在的她。对于我来说，我觉得她成绩很好，但是临近中考，焦虑困扰着她，她在尽力讨好所有人，在避免老师的批评，在因为选择题多错了几题而每夜睡不着觉。尽管我们两个家庭背景并不相同，我的家庭条件比她好得多，但我们却觉得同病相怜。


=========

五一去香港旅行，在动车上听到一位父亲在和儿子科普关于铁路知识。不得不感叹，我比这个小男孩多读了 10 年的书，却得不到他现在所拥有的家庭教育。而這樣的事，不僅僅發生在我一個人身上，還有很多很多的朋友，也是如此。


家庭的氛围，父母与孩子的交流，对孩子的关心，真的非常非常重要。对接的时候遇到一位朋友，与父母的交流少，父母对她的关心也不足，她的生活很压抑，休学在外自己住，住在一个小小的，没有窗子的房子里。她用那个房子困住了自己，但她也在挣脱这个无窗的房子，走向属于自己的世界。


还有一个也是对接认识的朋友，她一直在努力修復她與父母間的間隙。她真的很优秀，目前也在 Savus，为东亚式家庭的推广做出了很大的贡献。她在不停思考自己与父亲和母亲的关系，想和母亲关系慢慢变好，但又迷惑于为什么与父亲会慢慢疏远。她與父母的關係，雖然進展的速度很慢，但是她所作出的努力，是我們都能看得見的。人總是感性的，不善表達愛的父母也會為此觸動。我看到了我們共同所盼望的好結果，即將在她的身上發生。





给大家分享下我近期很喜欢的照片吧


///



不得不说这真是一大段失败的文字，但故事本身就是破裂的，破镜重圆总是有裂痕，东亚式家庭的负面伤害对于孩子而言不可能不留痕迹地抹去。


不管怎麼樣，我仍然愛著我的家庭，我的父母，他們也為我付出了很多。


希望我們都能夠有個好結局，希望了希望。
