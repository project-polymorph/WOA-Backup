心理学新书推荐


《向内看见》



出版时间：2023年1月

出版社：新星出版社

全书字数：15.1万字



《向内看见》这本书有4个部分，共8个章节。其中囊括了每个人的生命中都要经历的认识自己、认识家庭、了解人际关系、理解性别和亲密关系这四个重要课题。它仿佛沿着人的出生时间线，带领着我们以不一样的角度，又重新去看一些生活中我们习以为常的情况。





#1

认识你自己  


为什么有时候一件事情没做好，或者一句话没说对，大脑中就有个声音不停地责怪自己？

为什么一做决定，头脑中就浮现糟糕的后果，怎么都挥之不去？本来有充足的时间完成任务，可是自己非要拖延，每次都挤在截止时间前的几个小时去拼命完成？

这几天心情不好总想哭，我是抑郁了吗？一下班就想玩游戏，玩到半夜都不停，我会不会是游戏成瘾了呀…


如果你也遇到过类似的问题，那么这本书会从是什么、为什么和怎么办等多个方面，剥丝抽茧，让我们明白自己是怎么成为今天这样的人，以及如果对现在的状态不太满意，要如何去调整自己的思维和行动，变得对自己更满意一点，更爱自己一点。


《生命的重建》中有这样一段话：“这么多年，你一直在批评自己，可也没什么效果。试着认可自己，看看会发生什么。”如果你常常觉得自己不够好，甚至有些讨厌自己，不妨问问自己：我眼中的自己，真的就是全部的我吗？就像书中所说，对自己的讨厌，偶尔也可以动摇一下。


相关书籍推荐：

  1. 《生命的重建》 露易丝·海

  2. 《冒名顶替综合征》 米夏艾拉·穆逖兮

  3. 《恰如其分的自尊》 克里斯托弗·安德烈,弗朗索瓦·勒洛尔


#2

认识原生家庭


很多书籍和课程试图告诉我们，怎么培养出一个心理健康乐观开朗的孩子。然而这本书却反其道行之，专门用一个部分来告诉我们，如何培养一个“心理有问题”的孩子。例如“过度协助”，家长出于好意过度帮助孩子完成任务，这会导致孩子难以发展基本技能。

 

还有一个很多人都关心的问题：和父母的关系，会影响成年后和伴侣的关系吗？很大程度上，这个答案是肯定的。就像我们会发现，小时候受过家暴的孩子，长大后下定决心一定不会这样对待自己的孩子，可一遇到问题却还是忍不住动了手。我们或许也经常从伴侣身上，看到与自己父母相似的影子。即使同这个伴侣分手，找下一个，再下一个，却还是会反复陷入痛苦又无力摆脱的亲密关系。

 

心理学上把这称作强迫性重复。过去的创伤模式反复出现在我们面前，一方面折磨着我们，让我们感到难痛苦；另一方面也为我们提供了修复创伤的机会。真正决定我们生活方式的，并不完全是我们童年的经历，也取决于我们对自己经历的理解和认识。


今天又是多云，没有太阳，天不甚亮。我最喜欢大晴天的，窗帘拉开，阳光霎时就涌了进来：太阳的温度、太阳的味道，但也仅限于我一个人的时候才敢这么自由。

他人在的时候，我总疑心灿烂的阳光会惊扰了别人，所以我总是先偷偷地拉开半拉，看看阳光蔓延的范围，若是情况良好，便放任这半米阳光的照耀；若情况不妙，我只能在心里和阳光道个别，倒放般地合上“入口”。


相关书籍推荐：

  1. 《也许你该找个人聊聊》 洛莉·戈特利布

  2. 《如何说孩子才会听，怎么听孩子才肯说》 阿黛尔•法伯,伊莱恩•玛兹丽施

  3. 《原生家庭生存指南 : 如何摆脱非正常家庭环境的影响》 奥利弗·詹姆斯


#3

拨开人际关系中的迷雾



前段时间一部较火的电视剧《黑暗荣耀》，讲述了女主在学生时代被同学霸凌，长大后一步步谋划复仇的故事，引起了很多人的共鸣和讨论。


在《向内看见》中，作者从另一个角度，给想尝试从自身进行疗愈的人提供了一些建议：认知层面，牢记被霸凌不是自己的错；行为层面，可以向值得信赖的朋友、家人或者心理咨询师等寻求支持；还可以尝试和自己的情绪进行对话，找到另外的方式重新认识自己的情绪，慢慢地重建自我。

 

书中还对讨好、总是“背黑锅”、爱生闷气、习惯性说谎、道理都懂却还是过不好一生等现象进行了介绍，并提出了相应的解决办法。如果有感兴趣的话题，可以试着去书中进行更深层的探索。


相关书籍推荐：

  1. 《非暴力沟通》 马歇尔·卢森堡

  2. 《人间游戏 : 人际沟通心理学》 艾瑞克·伯恩


#4

性别认知与亲密关系  


书中有句话让我印象深刻：她的话一直被无视，直到他“父述”了一遍。

 

脱口秀演员颜怡颜悦在脱口秀大会第四季中也提到过类似的场景：在男性浓度过高的饭局，女性只有表达以下观点才受他们欢迎：“嗯”“是”“对”“哇偶~”“真的吗”“是啊对啊对啊是啊”“好有道理啊”。

 

职场女性歧视、男性凝视、被性别框架所束缚，这是我们不得不面对的社会现状。嫉妒、被背叛、感情中出现暧昧和备胎等，也是很多人为之苦恼的情感问题。本书的最后一个部分，就是从这些问题入手，帮助我们去觉察这些现象背后，隐藏了哪些未被看见的感受与需要。


相关书籍推荐：

  1. 《亲密关系》 罗兰·米勒

  2. 《煤气灯效应》 罗宾·斯特恩

  3. 《愤怒之舞：亲密关系中情绪表达的艺术》 哈丽特•勒纳

就像李松蔚在《向内看见》新书首发活动中所说：“对大多数人来讲。‘看见’其实是一个非常奢侈的体验。回想我们在日常的交流中，跟别人的关系里面，我们多数情况下是‘被要求’的。很少有人能在乎我们自己的内心感受和体验。”

 

人要被好好爱过，才有能力爱自己。所以向内看见，看得见自己，才能看得到别人；读懂了自己，才能尝试着去理解他人。拥抱内心那个或许曾经未被善待的自己，看见ta，像曾经希望被理想中的父母呵护那样养育ta，或许这就是离幸福越来越近的过程。
