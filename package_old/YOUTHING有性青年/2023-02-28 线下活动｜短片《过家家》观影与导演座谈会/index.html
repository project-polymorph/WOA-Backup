过家家海报（中文）


年轻的小林和更为年长的传玉在日常生活之余，从事福利姬的软色情行业补贴家用。她们在一起生活，既像姐妹又像情人。某天要求“线下定制”的男人M，雇佣她们扮演他为期一周的女儿和妻子。然而三人的关系在这段时间逐渐变得紧张，M最终在“扮演”中完成了谋杀妻女的剧情。扮演结束后，小林和传玉的关系似乎也发生了某种隐秘的变质，一种目不可见的虚假……她们是否还能回到原初的伊甸园呢？

曹昕怡

中国传媒大学影视摄影与制作专业在读。曾获上海作家协会文学新秀选拔总冠军。制作短片《如履薄冰》、《生吞》、《一次流产》、《后台对话》等。参与编剧的长片《流水无尽》，在剧情长片《常娥》担任摄影指导与编剧。



看上去其乐融融的丈夫妻子女儿，这就是所谓的幸福吗？


短片《过家家》入围英国leeds酷儿电影节、新加坡电影节酷儿单元、纽约reel sister电影节、上海国际短片周、北京国际短片周、北京酷儿电影节等等。


作者带着朋克式的反叛精神，从女性角度出发，用非主流反常规式的叙事手法，带领观众重新审视与思考家庭、权利、秩序对人的影响。当看上去和谐的关系某一天被打破之后，我们能够愿意去直面那些裂缝吗？
