 声援伊朗抗议足球员恐面临死刑 


随着近日伊朗国内民众示威抗议活动的继续，伊朗政府正计划新一轮的处决行动，潜在的处决对象至少43人，包括早前盛传被判死刑、该国年轻足球运动员Amir Nasr-Azadani。26岁的Amir Nasr-Azadani被当局指控杀害3名安全部队人员，因设计武装叛乱被捕。国际足球职业联会曾对此发声，表示震惊并呼吁撤销死刑判决。他曾效力伊朗顶级联赛球队。


此外，早前伊朗当局逮捕知名女性Taraneh Alidoosti引发国际声援。据阿姆斯特丹国际纪录片影展官网发布的消息，包括鹿特丹影展、柏林影展、德国电影学院及欧洲电影学院在哪的多国国际电影组织和机构共同发表声明强烈抗议伊朗当局的做法，并要求立即无条件释放她。


自伊朗22岁女子阿米尼（Mahsa Amini）因涉嫌违反头巾规定遭遭道德警察拘留期间死亡以来，伊朗全国持续出现民众示威抗议活动，截止12月初，在安全部队的镇压行动下，已有超过400名示威者死亡，至少182000人被拘留。


 塔利班禁止NGO女性员工于境内工作 


阿富汗塔利班政府向所有获得许可的非政府组织（NGO）去信，命令所有当地和国际的NGO禁止其女性雇员上班，若不遵守规定将吊销执照。塔利班经济部发言人证实了这份禁令的有效性，并表示「最近我们接收到关于NGO女性雇员不遵守伊斯兰头巾要求和其他伊斯兰酋长国法律法规的严重投诉，因此决定命令当地和国际非政府组织的所有女性雇员暂停工作。」联合国对此提出前侧，指禁令剥夺妇女选择自己命运的自由及基本权利，明显违反人道主义原则。联合国阿富汗问题特别代表表示，尽管联合国尚未收到该命令，但抗议预见与UN签约的非政府组织开展的大部分活动将受到严重影响。

 西班牙禁止玩具中出现性别刻板印象描述 


西班牙本月起实施新版「道德行为准则」（The code of conduct），自2005年以来睽违17年再次修订该准则，禁止玩具广告中出现带有性别刻板印象的描述。西班牙政府表示，希望玩具行销广告中的性别刻板印象走入历史，希望玩具制造商避免在广告宣传中加入性别刻板印象元素，例如，洋娃娃是专为女孩设计，模型更适合男孩等，消除社会大众对性别角色的偏见。政府声明指出，「认为女孩就适合会做家事的玩具，或男孩就适合科技产品的玩具，这类描述将被禁止使用」。新版「道德新闻准则」主要针对目标对象为15岁以下的青少年广告，尤其着重7岁以下儿童接受到的广告行销内容。

 美国约有120万对同性多元家庭 


根据美国普查局日前公布的人口统计数据显示，2021年约有120万对同性多元家庭，比2008年的54万户暴增120%，这120万对中，约有59%已婚，剩下的41%未婚。以同性多元家庭比例分布来看，美国首都华盛顿特区占2.5%，未全国最高。最低的是南达科他州，仅占0.4%。

 韩国男女工资差距逾31% 

 连续26年占OECD国家之首 


根据经济合作暨发展组织（OECD）最新发表的数据显示，2021年南韩男女工资差距高达31.1%，在39个OECD成员国中居冠，也是该国自1996年加入OECD以来连续26年男女工资差距排行第一。OECD的统计数据指出，去年南韩男女工资差距高达31.1%，其次为以色列的24.3%以及日本的22.1%。有专家指出，男女工资出现差距的原因在于女性职业生涯中断和论资排辈，在目前的工资体系中，只有在同一单位长时间上班，工资才能上涨。妇女在30多岁时因生育面临职业生涯中断，难以晋升为管理人员，女性在休完育婴假，返回职场时，其停职时间并不计入工龄，这导致男女工资出现差距。

 日本将废除女性离婚百日内禁止再婚规定 


日本自1898年起实施至今的民法「嫡出推定」制度，规定女性从离婚起算300天内生产的孩子，将被推定为前夫的小孩。外界认为，这项规定造成许多女性为了避免这种情况，不帮孩子办出生登记，进而产生无户籍儿童。日本政府为解决此问题而提出民法修正案，实施超过百年的「嫡出推定」制度进入获日本众议院修正通过，一方面维持女性离婚300天内所生小孩为前夫小孩同时，也增列如果女性生产时已再婚，所生孩子为现任丈夫小孩的规定，而现行民法中限制女性离婚百日内禁止在婚的规定也将一并被废除。


 美元纸钞上首度出现两位女性签名 


依照美国法律，任何在市面上流通的美元钞票，都需要有美国财政部长和财务长的签名，才被认可为合法货币。而此次新改版的美元钞票将会出现现任财政部长Janet Yellen，及新上任的财务长Lynn Malerba的签名。这次的签名将创下多个历史，不仅是美钞上首次同时由两位女性签名、首次由女性财政部长签名，也是首次每周原住民女性签名出现在美国货币上。Janet Yellen表示，财政部中女性劳动力比例达62%，女性员工在这个曾经由男性主导的领域取得重要进步，并肯定她们在财政部以及更广泛的经济学领域做出的贡献。「货币上的签名并不关于我，而是我们一同创造了更强大、更具包容性的经济。我希望今天能提醒人们，我们一直在名为平等与包容的道路上前进。」新版货币预计将在2023年初流通。
