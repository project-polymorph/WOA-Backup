#

声带手术适合怎样的人？

#


那今天我就向你们科普一下，其实声音低沉的人更适合学习伪音。这是因为声带手术往往对患者的先天嗓音要求极高，这个时候就要你本身的声音条件比较好。

在做声带手术之前，医生会对你的声音情况进行测试和评估，如果最终测试评估过后的结果不行，你也是不能进行声带手术的。这当中有一项就是看你的声音低不低沉，因为当你的音色过于低沉时，就算做了声带手术，术后的效果也并不理想。这个时候你就需要声音训练课程➕声带手术配合进行。
还有一点很关键的，我要提醒一下你们。带手术采取的原理就是将声带截短，达到提高声调的效果。但是你要知道除非你的手术非常成功，否则还可能面临着反弹的风险的！


但是如果直接选择通过伪音来改变声音，就不会再有这样的顾虑了。学习伪音是从音高频率上通过训练直接进行更高位置的声音状态发声，从而解决声音低沉手术不理想的情况。

这种无创的发声方式更容易让大家接受，也能更为有效的帮助所有跨性别者改变声音。


据不完全统计的问卷调查显示，大多数跨性别者用伪音来改变声音是更为合适的，但是大家通常认为通过伪音来改变声音非常的复杂和困难。其实只要有专业的老师，根据你的情况制定合适的课程来学习，坚持几个月你就能看见有效的变化。

而这样的方式跟伪音比起来风险更小，伤害也更小。不用等待漫长的声音恢复期，更不用担心声音反弹而每天提心吊胆。最重要的一点是你自己想要什么音色都是可以自由控制的，萝莉音御姐音或是自然的女生说话的声音，你都可以拥有，而且可以自由的控制。相对于声带手术来说，费用也是要低得多的。

所以说，声带手术更适合声音条件本就比较好的跨性别者，良好的嗓音再加上手术的加持，会对声音做到更好的改善。
那如果你先天的声音条件不太好，甚至比较低沉，那么则是伪音比较适合你的，这样的方法风险小，也能帮助你获取想要的声音。
