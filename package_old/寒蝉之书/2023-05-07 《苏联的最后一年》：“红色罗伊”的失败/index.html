《苏联的最后一年》：“红色罗伊”的失败

原创 十七年的寒蝉 寒蝉之书
2023年05月07日 00:55 北京

《苏联的最后一年》是罗伊·麦德维杰夫在2004年写成的书（因为他是苏联异见者中为数不多坚持列宁主义和苏维埃政权的人而被称为红色罗伊，他的名字也是为了纪念印度共产党的罗易，因为他的父亲是布尔什维克的活动家，后来死于大清洗），据说前后写了十几年之久。虽然顶着这样的名字，实际上这本书包括了戈尔巴乔夫时代的苏联政治事件的主要历史，并且涉及和点评了苏联中后期乃至于整个战后时期的一部分大事件。这本书作为史书的价值还是可以的，因为罗伊同时是苏联异见者、历史学家、亲历者和苏共最后一届的中央委员，使得他可以得知比较全面的消息，做出相对中肯的评价。并且，比起他在政治上的成名作《让历史来审判》（为了揭露斯大林时代的历史真相而写的与官方党史针锋相对的书，如果不熟悉那个年代的官方党史作为前提，就根本没办法阅读），《苏联的最后一年》相当的好读，而且翻译质量也不错。罗伊本人是苏联异见者左派的代表人物（中派/自由派代表是萨哈罗夫，右派是索尔仁尼琴，此外应该还有各民族的民族主义派，但相比之下很少被提到；此外，更“流行”的说法会把三派分别叫做社会民主派、新西方派和新斯拉夫派），笔者过去曾经很好奇，为什么中国的左派很少提到他，而自由派特别是多少有社民倾向的那些很喜欢说他，并且把他说成是品德堪比索尔仁尼琴的人物：在勃列日涅夫时期反对斯大林主义，在叶利钦解散苏共的时候是唯一一个发言反对的苏联人民代表，被秦晖说成是苏维埃的最后一个男儿——实际上罗伊是秦晖比较重要的思想来源之一。但从《苏联的最后一年》以及其他罗伊晚年的作品来看，至少在戈尔巴乔夫改革和苏联解体之后，和很多俄国知识分子一样，充满了悔恨和对中国改革开放的羡慕，以至于有些时候与普京、久加诺夫这种人可以混为一谈。或许他的品德可以作为信任的保证：我可以相信他不会在著作里故意撒谎；但是对于政治问题的判断，品德从来不是任何保证。

俄罗斯经常流行这样的话语：苏联的经济是很成功的，赫鲁晓夫—勃列日涅夫前期的苏联经济几乎是俄罗斯历史上最好的，人民的生活水平逐渐追赶西方；而且实际上在1991年苏联的经济都没有彻底崩溃，直到休克和大规模私有化才导致了独联体国家的贫穷。在俄罗斯以外，左派的朋友也经常听到类似的说法，这样的说法自然不很错，但是实际上忽略了很多重要的真相。尽管在1946、1947年之后苏联就没有出现大的经济危机，直到80年代开始发生缓慢（但不是灾难性）的经济衰退，但实际上这也是对于其中的一系列局部危机进行反应和改革的结果，每当苏联经济发生一定的困难，苏联官方都会启动新的新经济政策，只是程度上大小有别。赫鲁晓夫改革是这样，柯西金改革也是这样，只是他们都只至多允许小生产和国有部门的一定程度的自负盈亏这样的半市场化，但是反对私营经济，而甚至斯大林时代就因为战争而放开过私营企业的成立。因此，固然整体上说苏联经济是成功的，但它在成就阴影掩盖下的巨大弱点：高积累低消费、在农业上的落后、在经济总量迅速增长的同时在劳动生产率和技术革命上显著落后于西方甚至日本（苏联的汽车生产线甚至都是在关系缓和的时候从西方引进的，而且在汽车普及上不如美国、纳粹德国甚至今天的中国，苏联的工人阶级可没有五菱宏光），荒诞性的僵化——在苏联开办新工厂来生产牙膏或者女袜甚至需要政治局开会讨论，以及最要命的——每当遇到一定规模的麻烦就只能想到通过局部的半市场化和引入一定资本主义因素来改良，这些都是我们需要认识到的。而且实际上类似的逻辑在东欧（尤其是匈牙利）和中国也存在。至于80年代的经济，固然没有90年代那么可怕，但是的确艰难到了一般资本主义国家经济危机的地步。

戈尔巴乔夫上台之后一开始只是表达“不能再这样生活”的意思——这是对于勃列日涅夫停滞时代的自然反应，而且戈尔巴乔夫原属于安德罗波夫的派系，本来就是在勃后期兴起的改革派成员——但他本身还没有市场化的打算。实际上，戈尔巴乔夫改革的开篇是禁酒（背景是苏联后期开始，在女性死亡率基本稳定不变的情况下，男性死亡率逐渐抬升，这被认为和酗酒有关。顺便一说，当我们说叶利钦时代死了几百万人的时候，实际上绝大多数也是男性——这说明叶利钦时期的经济破坏并不是饥荒那种类型的，而是更微妙的形式。至今俄国的男性平均寿命比女性短大约十岁，比“正常情况”的五岁左右差了很多，这一点而不是战争才是独联体国家女多男少的主要原因）和打击“非劳动收入”，后者指的是苏联时期各种影子经济。苏联的影子经济据估计大概有按GDP核算的苏联经济的15%左右，其中包括真正的贪污腐败，但是也包括各种不合法的和半合法的地下经济，包括把自留地的产出放到市场上卖和酿造私酒这样的小生产以及打零工在内。苏联异见的诗人、作家布罗茨基就是以这类“寄生虫”的名义审判的，但这只是口袋罪。在赫—勃时期乃至于斯大林时期的很多时候，影子经济被默许甚至鼓励，因为甚至工人阶级都需要依靠它来作为收入和消费品的补充，打击影子经济会造成经济和生活水平的震荡；而戈尔巴乔夫上台之后就很快开始严打影子经济。戈尔巴乔夫走向市场经济继而走向政治改革（公开性、历史真相和与自由派和解）并不是由于自己的判断（实际上罗伊相信戈尔巴乔夫能力低下根本不知道要做什么），而是因为苏联已经存在了小半个世纪的、实际上就连勃列日涅夫也承认的、而且在其他社会主义国家也很大程度上存在的共识，差别仅仅在于引入资本主义到什么程度而已。苏联的保守派不愿意和戈尔巴乔夫那样进行到允许生产资料私有制普遍归来的地步，但是实际上根据当时的民调，即便是很多群众都愿意保卫苏联和社会主义，但是极少人反对任何形式的私有制；而罗伊本人也逐渐认同了这种倾向。实际上如果社会主义国家持续性地把引入资本主义当作应对经济困难的唯一办法的话，那么走向恢复私有制也只是时间问题，这当然不是宿命论地说社会主义的命运在赫鲁晓夫时期就写好了，而是说如果无法提出非资本主义而行之有效的方案（“割资本主义尾巴”显然只是有害经济的运动群众），那么就不可避免地走上这条道路，所以在苏联、捷克斯洛伐克，特别是南斯拉夫和匈牙利实行的改良政策就不可避免地在后来的其他地方变成了真正的资本主义，而在90年代抵制资本主义的古巴发现如今自己别无选择。这种走向资本主义的改革绝不是因为个别人物的历史性恶意，而是社会主义面对自身失败的某种无能，跟在80年代的中国身后以更大的步伐开始改革的苏联也是这样，只是它采取了应激性的、毁灭性的形式，而古巴今天在采用更合理的形式而已。

如此一来，作为改革派（毫无疑问当时的苏联需要巨大的改革）的罗伊支持和羡慕中国改革就不再奇怪，因为他作为苏共干部和知识分子中最具有自由人道思想和政治勇气的那一个，固然走上了政治异见者的道路，但归根结底是苏共干部和苏联知识分子这两个阶层的交叉性代表，他完全认识到这两个阶层所共识的危机即上述社会主义国家的危机，提出的方案也不异于流行的方案，一切都只是程度的区别。或许在80年代末期和90年代的社会主义做出的选择超过了罗伊本身希望接受的程度，但这却与他的思想在相同的方向上。而且在当时，他显然陷入到不知如何是好的情况：作为知识分子他显然认同自由派和右派（当时很难说已经分化）对于人道主义和历史真相的诉求，这样的事情显然也是苏联应该做的；作为苏共的成员和社会主义者他希望保卫苏维埃制度、列宁的党和社会主义的荣誉；而作为聪明的学者、苏联人民代表和苏共的中央委员他显然已经知道苏联的政治经济问题的急迫。所以他的立场和戈尔巴乔夫一样，试图在改革的同时保卫苏联，这使得他很多时候简直成为了苏共保守派和自由主义活动家的某种折中形式：我们要自由，但是要慢一点；我们要历史真相，但是请不要颠覆苏联；我们要市场经济，但是请继续保持苏维埃的红旗；我们把一切怪给斯大林，但是要尊重列宁的荣誉（先不要说在巨大的历史意识形态包袱之下这一套本身就竞争不过自由派和右派的“红色魔鬼学”，这种类似于赫鲁晓夫的做派在自由派已经公布了更多的历史真相之后显然难以说服群众）。在这种矛盾之下，固然有了自由的社会政治条件，甚至有民选的和苏共的双重政治身份，但他的政治活动却几乎停滞了，他没办法和自由派那样脱离苏共而行动，因为这是摧毁苏维埃的破坏行为，也不乐意和苏共保守派一样攻击民主派，因为他们关于自由、民主、历史真相和公开性的呼声是合理的。尽管作为苏联三大异见者的罗伊当时是唯一一个能活动的（萨哈罗夫去世了，索尔仁尼琴还在国外），但是他放弃了活动，几乎只作为观察者和议员存在。实际上，他唯一的政治活动就是在苏共被解散之后建立了“劳动人民社会党”并维持苏共的最后一个党纲，也就是作为社会民主主义和民主社会主义之间的一个党而存在，这个党没有取得什么成果，但却在1993年帮助了久加诺夫的俄罗斯共产党的重建。后者虽然几乎举着斯大林主义的旗号，但是是什么性质已经不需要多说了，俄共如今只是俄国当局可笑的点缀，而且在纲领上早已经是支持中小私有企业的。假如久加诺夫真的在1996年战胜了叶利钦的话，或许能停止不负责任的私有化来减轻俄罗斯民众遭受的痛苦，但长期来看恐怕很难说比后来的普京当局更好。

由此看来，我们不得不承认这样的事实：“红色罗伊”或许在历史问题和揭露苏联官方的谎言上有一定的贡献，但是他在性质上（特别是在80年代之后）和戈尔巴乔夫甚至久加诺夫没有本质的差别，他们的差别仅仅在于是否有良心，至多相当于萨哈罗夫之于叶利钦和索尔仁尼琴之于普京那样，而正如之前所说良心根本不是政治道路的保证。除了在人道主义方面有更多诉求之外，苏联的绝大多数社会民主派和共产保守派一样，他们认同的社会主义仅仅是红旗、苏维埃国家和列宁的名誉，他们认同共产主义仅仅是因为列宁时期的革命者和斯大林时期的祖国保卫者“很伟大”，甚至仅仅是因为生于红旗下的祖国热忱，他们就像非社会主义国家的一些右派一样，只是由于祖国和先辈的历史荣耀而保卫祖国的，差别仅仅在于这是一个被涂红了的“社会主义祖国”，他们理解的社会主义除了苏维埃政权之外最多就是人道主义、生产力进步和人民生活条件改善这些社会民主主义甚至多数自由派都认同的东西，而反对资本主义不过是反对贫穷和过度贫富分化甚至仅仅是反对资本主义-帝国主义的西方！他们没有任何的最基本的无产阶级的意识，甚至不是和今天中国网左那样由于对现实的陌生、行动的困难和作为知识分子的傲慢而在实践上脱离无产阶级群众，而是在理论上就完全没有任何对群众的理解和认识。那时的苏联和之后的俄罗斯有足够的行动自由，而且苏联劳动者的受教育程度极高，但凡他们有点这个意识都应该知道要做什么；但他们意识里的无产阶级有时候只是一个抽象概念，有时候从属于人民或者人民群众这样的抽象概念，有时候甚至都不存在——而这种脱离群众又和苏共的逐渐脱离群众是一脉相承的。在80年代的苏联一切都已经太晚了，这个社会主义国家已经无法拯救，这时候唯一有意义的行动不是保卫红旗——这倒不是因为像某些自由派说的那样，苏共名誉极差，甚至可能从来都是一个犯罪组织，而是因为苏共和苏维埃国家已经是无可救药的组织——当时部分自由派甚至提议像纽伦堡审判一样审判苏共，固然左派都会觉得不公平，但是如果真的对苏共势力进行清算和排除，就不会导致久加诺夫和普京的出现，甚至不会导致叶利钦本人的政权的出现，政权会像东欧那样落到诸如索布恰克那样的体制外自由派身上，无论如何都比叶利钦好太多了——所以只有和自由派那样脱离它才有可能进行真正的活动；也不可能和自由派一样重新提“一切权力归苏维埃”的口号，因为这时候的苏维埃哪怕不再是橡皮图章的样子货议会，也最多成为“正常的”资本主义的议会。这时候唯一可能做的事情，就是和地区性和全国性的罢工结合，将当时在部分地区（主要是矿区）已经承担起来部分管理功能的罢工委员会转化为常态性的工会乃至于地方政府，告诉工人们现在几乎崩溃的苏联国家不可能（甚至已经不是是否愿意的问题）满足工人哪怕合理的和基本的需求，只能通过在当地尝试转产轻工业的消费品和农业产品，和当地的农业部门合作并且鼓励甚至动员小生产这样的方式来自己动手丰衣足食——一句话，就和苏联遍地成立的民族主义分离地区一样，将这些罢工的地区转化为半独立的政权，准备在苏联崩溃的遗产上争取空间。这很难，比当时苏联的自由派建立全国性组织这样的行动要复杂得多，甚至超过了波兰团结工会的事业，但是几乎是在末期的苏联保卫社会主义的唯一办法。我不应该幻想这样的事情能够完成，但是罗伊大概连想都没想；他想的大概只有向苏共保守派说不要迫害自由派，和自由派说不要攻击我们的联盟，或者是怎么在不改变颜色的情况下走向改革这样的事情，这样的派别被自由派和右派彻底击败简直是再自然不过的。

当然，在苏东存在很多左翼名义下的反抗活动，我并不是说所有这种活动都是同一性质的，但是在80年代发生的事情已经很不一样了。因为社会主义阵营不是在1991年失败的，也不是在1985年失败的，而是最迟在1968年或1975年就已经失去了生机，不仅苏东的官方变成了保守的、陈旧的垃圾当局，而且这些国家的左翼反对派也失去了活力。如果在波兰等其他国家表现为工人群众也不支持左派的观点而支持右派的观点（这对于波兰后来成为一个西方阵营的右翼堡垒有很大的关系）这样的事情，那么在苏联就表现为左翼反对派退化成了官方保守派和反官方自由派的浆糊、社会民主党和帝国主义者的杂烩，甚至日后成为醉鬼的叶利钦都知道自己应该做什么的时候，他们却不知道自己当下的事业而白白蛰伏浪费时机。这种不知道自己应该做什么的困境一直持续到后来的俄罗斯。实际上，罗伊还写书支持过普京。相比索尔仁尼琴支持普京至少是因为他们都是支持民族精神特殊性的右派，那么罗伊完全是保留了自己在80年代那种胡思乱想的传统，作为一个老头子他没办法甩掉巨大的意识形态历史包袱。实际上罗伊至今97岁，仍然活着（这让我很惊讶），但就像久加诺夫仍然活着而戈尔巴乔夫刚刚才去世一样，他们都早就是被历史遗弃的死人了。
