驼峰鼻/鹰钩鼻矫正整形手术

手术目的与效果：

改善驼峰鼻，柔美、女性化


手术方法：

1、单纯驼峰鼻矫正：凿除或磨除凸起的驼峰


2、鹰钩鼻矫正：鼻尖上旋


临床案例：

案例1：女，24岁，驼峰鼻、鼻尖下垂、软骨性歪鼻，予驼峰截除、鼻尖上旋、软骨性歪鼻矫正


术前注意事项：

  1. 无出血性疾病以及心肺肝肾等重要器官活动性或进行性疾病，如患有高血压、糖尿病或传染性疾病等患者需在病情控制平稳后手术。

  2. 长期服用类固醇激素和阿司匹林等药物者，最好能在术前一周停用，请咨询相关医生。

  3. 如有以下鼻部情况者不建议手术：鼻部皮肤存在疖、毛囊炎等感染灶，急性鼻窦炎、过敏性鼻炎发作期、3个月内有急性鼻外伤史等。

  4. 女性患者手术请避开月经期；妊娠前期（3个月）和妊娠后期（3个月）暂缓手术。

  5. 因术后局部轻微肿胀可能延续数周，此时鼻部形态略欠自然，故手术安排时请考虑在此段时间内避开重要的社交场合。

  6. 因鼻骨截骨手术过程以及术后恢复较为复杂，有时难免能精确掌控，不除外最终有一定程度的两侧不对称或者局部不平整，需要半年后再次修复可能。


麻醉方式：

  • 局部麻醉：手术部位局部注射麻醉剂，不另收费。

  • 静脉全麻辅助：手术时间长、部位多、或对疼痛耐受差者，静脉给药，无需气管插管，另收费（根据手术时间长短1500-3000元不等）。


术前准备：

  1. 常规检查：(可手术当天做，局麻者无需空腹，静脉全麻者空腹化验；外地患者为节省时间也可以先在当地三级医院提前完成，3个月内有效)

    局麻手术者：血常规、尿常规、凝血四项、美容免疫四项（乙肝、丙肝、梅毒、艾滋）；

    静脉全麻手术者：上述6项、肝肾功能、空腹血糖、心电图、胸片

  2. 选择局麻者正常饮食，选择静脉全麻辅助者术前6小时禁食、4小时禁水

  3. 术前注意术区清洁，剪短鼻毛与胡须，勿化妆

  4. 准备术后用药：棉球若干、消肿药（1周量）、止疼药（1-2次）


术后注意事项：

  1. 术后鼻孔内填塞的纱条第二天可自行摘除；如有外用固定装置7天后摘除。

  2. 伤口避免沾水，鼻部避免用力擤鼻和揉搓，避免弯腰低头。

  3. 术后当天勿饮热饮，一周内忌烟酒、鱼虾、辛辣刺激食物。

  4. 口服抗生素3天预防感染，必要时可服用止痛药（如去痛片、曲马多等等）。

  5. 鼻小柱切口缝线术后7天拆线（或遵医嘱），鼻孔内可吸收线可拆可不拆。

  6. 术后术区出现水肿、青紫、淤血、少量渗血、鼻塞、鼻孔暂时不对称、取骨区疼痛等，属于正常现象，大部分在1-2周后逐渐消退，如发现偏斜明显、肿胀淤血或疼痛加剧、持续影响通气等等，请立即联系医生。

  7. 3个月内避免戴框架眼镜，鼻部禁止揉扪及外力碰撞。

  8. 半年后请联系医生复诊。



鼻基底填充手术

手术目的与效果：

填充鼻翼或鼻小柱基底（“富贵术”），改善面中部凹陷，间接改善嘴突印象。


适应证：

侧面观上颌骨前突、鼻基底或面中部塌陷，又不接受正颌手术者。


手术方法及图示：

口内切口，假体（建议膨体）填充鼻小柱基底或者双侧鼻翼基底。


临床案例：

案例1：女，22岁，上颌前突，中面部鼻基底凹陷。予硅胶假体填充鼻基底，术前与术后半年。


术前注意事项：

  1. 无出血性疾病以及心肺肝肾等重要器官活动性或进行性疾病，如患有高血压、糖尿病或传染性疾病等患者需在病情控制平稳后手术。

  2. 长期服用类固醇激素和阿司匹林等药物者，最好能在术前一周停用，请咨询相关医生。

  3. 如有以下唇部情况者建议暂缓手术：唇部存在口腔溃疡、毛囊炎等感染灶，鼻窦炎急性期等。

  4. 女性患者手术请避开月经期；妊娠前期（3个月）和妊娠后期（3个月）暂缓手术。

  5. 如果上颌前突严重者，应首先考虑可以先考虑到口腔科进行正颌手术，效果更佳。

  6. 风险告知：假体填充可能会存在移位、穿透、感染等风险，一旦发生需要二次手术取出。


麻醉方式：

  • 局部麻醉：手术部位局部注射麻醉剂，不另收费。

  • 静脉全麻辅助：手术时间长、部位多、或对疼痛耐受差者，静脉给药，无需气管插管，另收费（根据手术时间长短1500-3000元不等）。


术前准备：

  1. 常规检查：(可手术当天做，局麻者无需空腹，静脉全麻者空腹化验；外地患者为节省时间也可以先在当地三级医院提前完成，3个月内有效)


    局麻手术者：血常规、尿常规、凝血四项、美容免疫四项（乙肝、丙肝、梅毒、艾滋）；


    静脉全麻手术者：上述6项、肝肾功能、空腹血糖、心电图、胸片、胸片


  2. 选择局麻者正常饮食，选择静脉全麻辅助者术前6小时禁食、4小时禁水

  3. 术前注意术区清洁，剪短鼻毛与胡须，勿化妆

  4. 准备术后用药：消炎药（3天量）、止疼药（1-2次）、漱口水（1周量）


术后注意事项：

  1. 术后伤口避免沾水，鼻基底部避免用力揉搓，避免弯腰低头。

  2. 术后前3天进软流食，避免过热食物，进食后注意漱口。1周内唇部避免夸张表情，尽量少说话，避免大笑以及用力哈欠。可以刷牙，但注意轻柔，勿触碰伤口。

  3. 术后1周内忌烟酒、鱼虾、辛辣刺激食物。

  4. 口内缝线为可吸收线，可不拆，1-2月左右自行脱落；如觉得不适也可以术后7天以后拆除。

  5. 术后口服抗生素3天。

  6. 术后术区出现水肿、青紫、淤血等，属于正常现象，一般在术后2-3天时水肿最厉害，可服用消肿药。大部分在1-2周后逐渐消退，如发现肿胀淤血或疼痛加剧、出血等等，请即联系医生。

  7. 术后可能出现唇部局部麻木或感觉迟钝，一般半年左右逐渐恢复。

  8. 最终效果可在术后3月以后再评价，如果存在局部矫正不足或矫正过度、不对称或者局部不满意可在3个月后再次修复。

  9. 为避免假体移位、穿透，3个月内尽量避免夸张表情。一旦出现假体移位、穿透、感染等，请立即联系医生处理。

  10. 半年后请联系医生复诊。

鼻孔/鼻翼/鼻柱整形手术

手术目的与效果：

改善鼻翼、鼻小柱外观精致程度、提升气质


术式选择：

1、鼻翼缩窄术：适用于鼻翼过宽、鼻孔“外张”者


2、鼻翼前缘下降术：适用于鼻翼前缘向头侧异位、鼻孔相对“朝天”者


3、鼻小柱上提术：适用于鼻小柱根部向下外露过多者


4、鼻小柱下延术：适用于鼻小柱退缩、鼻小柱上唇夹角过锐者


临床案例：

案例1：女，32岁，诉鼻翼前缘过高，正面看露鼻孔。手术予鼻孔前缘下降。术后1月照片


案例2：女，20岁，鼻翼缩窄。


案例3：鼻小柱根部提升

女，24岁，诉鼻小柱根部脱垂，侧面观鼻小柱-上唇角呈钝角。

鼻孔内切口鼻小柱根部提升，术前与术后1月，可见鼻小柱-上唇角恢复90度左右


案例4：女，22岁，上颌前突，中面部鼻基底凹陷。予硅胶假体填充鼻基底，术前与术后半年。


案例5：女，26岁，鼻翼缩窄+唇部综合，术前与术后2年。鼻翼缩窄+上唇人中短缩+自体脂肪注射丰唇（1次）+自体脂肪隆下巴，先后3次手术完成。

  1. 无出血性疾病以及心肺肝肾等重要器官活动性或进行性疾病，如患有高血压、糖尿病或传染性疾病等患者需在病情控制平稳后手术。

  2. 长期服用类固醇激素和阿司匹林等药物者，最好能在术前一周停用，请咨询相关医生。

  3. 如有以下鼻部情况者不建议手术：鼻部皮肤存在疖、毛囊炎等感染灶，急性鼻窦炎、过敏性鼻炎发作期，鼻部3个月内有过外伤史等。

  4. 女性患者手术请避开月经期；妊娠前期（3个月）和妊娠后期（3个月）暂缓手术。

  5. 鼻孔鼻翼手术通常采用开放式手术入路，需要在鼻翼处切口，术后1-2月内该切口略红稍硬，3月后逐渐淡化软化，谈话距离一般不易发现，但离近看仍可见痕迹。

麻醉方式：

  • 局部麻醉：手术部位局部注射麻醉剂，不另收费。

  • 静脉全麻辅助：手术时间长、部位多、或对疼痛耐受差者，静脉给药，无需气管插管，另收费（根据手术时间长短1500-3000元不等）。


术前准备：

  1. 常规检查：(可手术当天做，局麻者无需空腹，静脉全麻者空腹化验；外地患者为节省时间也可以先在当地三级医院提前完成，3个月内有效)


    局麻手术者：血常规、尿常规、凝血四项、美容免疫四项（乙肝、丙肝、梅毒、艾滋）；


    静脉全麻手术者：上述6项、肝肾功能、空腹血糖、心电图、胸片


  2. 选择局麻者正常饮食，选择静脉全麻辅助者术前6小时禁食、4小时禁水

  3. 术前注意术区清洁，剪短鼻毛与胡须，勿化妆

  4. 准备术后用药：棉球若干，止疼药（1-2次），外用祛疤膏（3月量）


术后注意事项：

  1. 术后伤口避免沾水，鼻部避免用力擤鼻和揉搓，避免弯腰低头。

  2. 术后鼻孔内填塞的纱条第二天可自行摘除；切口拆线一般7天

  4. 如无特殊交待不必服用抗生素，必要时可服用止痛药（如去痛片、曲马多等等）。

  5. 术后术区出现水肿、青紫、淤血、少量渗血、鼻塞、鼻孔暂时不对称、取骨区疼痛等，属于正常现象，大部分在1-2周后逐渐消退，如发现肿胀淤血或疼痛加剧、持续影响通气等等，请立即联系医生。

  6. 如消肿后出现切口瘢痕明显、鼻孔不对称等，请先耐心等待3个月，通常会逐渐变淡恢复，如3个月后仍不满意可以进一步修复；稳定后如果切口遗留痕迹可以进一步激光治疗减淡。

  7. 半年后请联系医生复诊。


鼻背鼻尖整形手术

手术目的与效果：

改善鼻背高度、鼻尖高度及突出度、细节精致化、提升气质。


手术项目选择：


切口选择：


鼻部填充材料选择：

  1. 硅胶假体：无毒、不致癌、不致畸，价格便宜，放置、更换或取出容易，但质地略硬，有移位、顶破皮肤可能，抗感染能力略低。目前可用于鼻背与鼻根部、鼻基底部位的充填。

  2. 膨体（ePTFE）：无毒、不致癌、不致畸，质地适中，可与周围组织“融”为一体，但价格较贵，有感染以及排异的风险，一旦出现通常需要手术取出。目前可用于鼻背与鼻根部、鼻基底部位的充填。

  3. 自体肋软骨：无毒、不致癌、不致畸，无排异反应，可与周围组织融为一体，但需同时行手术切取自身肋软骨。可用于鼻背与鼻根部、鼻尖部以及鼻基底部位的充填。

  4. 玻尿酸（透明质酸）注射：无毒、不致癌、不致畸，手术简单痛苦少，但效果仅维持半年到一年左右。


鼻尖填充材料选择：

  1. 耳软骨：可以轻度抬高鼻尖，但无法延长鼻尖，可用一侧鼻孔内切口手术，耳部需要取软骨，轻微疤痕

  2. 鼻中隔软骨：可以轻度抬高、延长鼻尖，但可能幅度有限，从鼻中隔中取软骨，需要做两侧鼻孔及鼻小柱切口，但不额外增加身体其它地方切口

  3. 肋软骨：可以最大程度抬高、延长鼻尖，效果确切，需要从胸部或乳晕做切口取骨


临床案例：

案例1：女，26岁，鼻梁塌陷，鼻尖表现点不突出。

手术方案：膨体隆鼻背，自体中隔软骨抬高缩窄鼻尖。

术前与术后1月照片


案例2：女，21岁，鼻背低平

手术方案：单纯硅胶假体隆鼻背

术前与术后4月照片


案例3：玻尿酸注射隆鼻（术前与术后即刻）


案例4：肋软骨综合隆鼻（经乳晕切口取肋软骨）

肋软骨综合隆鼻：自体肋软骨分体式移植物，隆鼻背+抬高延长鼻尖

微痕法取肋软骨：右侧乳晕切口取第5肋软骨，切口局限在乳晕边缘，相对胸壁切口要隐蔽的多。

女，28岁，术前与术后1月。


案例5：膨体隆鼻背+上唇人中短缩术


案例6：硅胶假体隆鼻背+自体耳软骨隆鼻尖。女，22岁，术前与术后2年，先后还进行重睑、内眦开大、自体脂肪面部综合填充、多次肉毒素注射瘦脸。


术前注意事项：

  1. 无出血性疾病以及心肺肝肾等重要器官活动性或进行性疾病，如患有高血压、糖尿病或传染性疾病等患者需在病情控制平稳后手术。

  2. 长期服用类固醇激素和阿司匹林等药物者，最好能在术前一周停用，请咨询相关医生。

  3. 如有以下鼻部情况者不建议手术：鼻部皮肤存在疖、毛囊炎等感染灶，急性鼻窦炎、过敏性鼻炎发作期，鼻部3个月内有过外伤史等。

  5. 鼻部手术以及术后恢复过程相对复杂，术前需要充分沟通，根据求美者自身鼻部条件设计合适的术式，并且交代清楚能够达到的效果。同时，术后也有可能会因各种原因出现假体或软骨移植物偏斜、移位、感染、改善不足、继发不对称、效果不满意等风险，一旦出现，潘柏林医生会尽一切可能补救，但需要您的理解和配合。

  3. 术前注意术区清洁，剪短鼻毛与胡须，勿化妆

  4. 准备术后用药：棉球若干，消炎药（3-5天量），止疼药（1-2次），外用祛疤膏（3月量）

术后注意事项：

  1. 术后伤口避免沾水，鼻部避免用力擤鼻和揉搓，避免弯腰低头。

  2. 术后鼻孔内填塞的纱条第二天可自行摘除；如有外用固定装置7天后拆除。切口拆线一般7天

  3. 术后当天勿饮热饮，一周内忌烟酒、鱼虾、辛辣刺激食物。

  4. 口服抗生素3天预防感染，必要时可服用止痛药（如去痛片、曲马多等等）。

  5. 如行肋软骨切取术，术后胸部疼痛2天左右，建议卧床休息。

  6. 术后术区出现水肿、青紫、淤血、少量渗血、鼻塞、鼻背不平整、鼻孔暂时不对称、取骨区疼痛等，属于正常现象，大部分在1-2周后逐渐消退，如发现鼻梁假体偏斜或移位，肿胀淤血或疼痛加剧、持续影响通气等等，请立即联系医生。

  7. 3个月内不戴有框眼镜，鼻部禁止揉扪及外力碰撞。

  8. 半年后请联系医生复诊。
