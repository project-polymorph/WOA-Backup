激素治疗 女→男（FTM）

原创 潘柏林医生团队 I 本来模样 I
2023年10月04日 16:45 北京

0

1

治疗目的


  • 帮助减少女性第二性征，促进/维持男性第二性征表达，体验所认同性别（男性）的生理状态，帮助自我接纳；

  • 如果已经接受性别重置手术，维持术后认同性别（男性）的基础性的雌激素水平，减少性激素不足相关症状

0

2

前提条件


  • 三级医院精神科或心理科医师开具的易性症（性别焦虑，或性别不一致）诊断证明；

  • 心理疏导、化妆、异装、美容等不能缓解性别焦虑，本人具有自发、明确、强烈的激素治疗诉求；

  • 治疗前筛查无激素治疗的禁忌症；

  • 年满18周岁，或年满16周岁且父母双方已书面知情同意；

  • 目前和将来无自然生育的诉求，或有需求者治疗前先进行生育力保存；

  • 能够规律地进行医疗监测和随诊。

0

3

使用药物


雄激素：可有口服、凝胶、注射等剂型。可以先从起始量开始，每三个月根据效果以及复诊检查结果进行调整

0

4

禁忌症


  • 对药物过敏者

  • 肝肾功能不全患者者慎用

0

5

启动治疗前检查


血常规、肝肾功能、电解质（含钙磷）、血糖、血脂、凝血功能（含D二聚体）、甲状腺功能、甲状旁腺素、25羟维生素D、性激素六项、骨密度检查、乳腺B超、妇科B超、腹部B超。

0

6

治疗中监测


  • 每三个月一次：血常规、肝肾功能、电解质（含钙磷）、血糖、血脂，性激素六项；

  • 每年一次：同“启动治疗前检查”；

  • 治疗前检查发现异常项目者，必要时需请相关专业会诊，加强监测。

疗程中注意事项


  • 少油饮食，尿酸升高者减少海鲜与啤酒摄入；

  • 适量运动与晒太阳，低骨量或维生素D缺乏者在医生指导下补充VD；

  • 如出现痤疮者，可以外用抗生素或维甲酸软膏。

0

9

可能出现的并发症与风险


  • 情绪波动；

  • 红细胞增多；

  • 体重增加、水肿；

  • 痤疮、油脂分泌增多、秃顶、睡眠呼吸暂停；

  • 糖尿病、高血压、高脂血症、高尿酸血症、心血管疾病风险增加；

  • 乳腺、宫颈、子宫、卵巢癌症、肝癌风险增加；

  • 卵巢形态、功能异常，不孕；

  • 肝功能受损；

  • 骨质疏松；

  • 激素治疗也可能无法完全缓解性别焦虑状态，最终仍需要求助于其它替代治疗方案可能。
