[招募]跨性别在线志愿者招募

跨性别在线 跨性别在线
2019年10月05日 00:00


        大家好，我是跨性别在线公众号的负责人，现在网站与公众号都是由我自己维护。因为各种个人原因前段时间停更了好久~为了给大家提供更多更好的内容，现开始招募一些志愿者，来和我一起维护公众号与网站，进行信息资料的收集、公众号编辑等工作。因为刚刚起步所以没有福（gong）利（zi），没有强制工作时间要求

 

希望招募

 

编辑组：2人

技术组：1人

翻译组：若干

联络组：1人

资源组：2人

个案组：2人

 

具体工作与要求见下方

 

总体要求

 

不限性别认同、不限性别表达

不限指派性别、不限性取向

学历、专业、经验

满足各部门的要求

认同跨性别

愿意帮助跨性别社群

有关于跨性别的基本概念的了解

能长期进行活动（3个月以上）

愿意帮助跨性别群体


 

编辑组

 

职能

负责编辑、修改公众号内容

负责排版公众号文章与翻译后的手册、论文

要求

熟练使用Word

能够进行简单的图片处理

加分项

有过公众号排版经验（公众号内容更美观）

能够使用专业排版工具进行排版（排版效率高）

 

技术组

 

职能

负责网站的维护

负责将公众号中的内容发布到网站

负责文档库在线阅读版本的创建

负责云服务器的维护

要求

能够操作Linux系统

会进行nginx与httpd下的SSL部署

会简单的静态网页制作。

加分项

有过wordpress建站经验（因为网站使用的是这个）

使用过gitbook或其他html书籍生成工具（因为在线阅读用的这个）

熟悉php（因为整体用的是php）

开发过小程序（想要进行小程序的开发）

 

翻译组

 

职能

负责翻译各类外国文献

要求

英语水平能够在词典辅助下阅读并翻译论文

加分项

有留学经历（对语言更了解）

有公认英语水平证书或成绩（可以判定英语能力）

会英语外的其他语种（可以翻译其他语言文献）

有过翻译经验（翻译效率会比较高）

医学专业出身（因为医学名词较多）

 

联络组

 

职能

负责与其他公众号、组织联络

负责请求其他公众号的授权

要求

无特殊要求

加分项

有社群工作经验（便于与其他组织沟通）

 

资源组

 

职能

负责收集各种有价值的文献

负责收集其他团体、组织的公众号的跨性别活动信息

负责收集其他组织的研究内容

要求

有检索与阅读中外文献的能力

加分项

在校学生（文献阅读较多、校内可以免费查看）

有社群工作经验（便于查找资料）

了解社群内的组织（便于了解与收集信息）

 

个案组

 

职能

负责跨性别咨询

负责跨性别求助

负责个案的转介

要求

无特殊要求

加分项

有社群工作经验（能够更好的了解转介目的地）

有心理学、法学背景（因个案可能涉及这些方面）
