# 什么是紫色纪念日 #


每年，因为遭遇来自于不同领域的欺凌，无数性少数青少年选择以自杀的方式结束自己的生命。为了唤醒人们对这一问题的认识，以及纪念因遭受网络欺凌而自杀的少年泰勒·金文泰(Tyler Clementi)，2010年“紫色纪念日”被创立。现在，“紫色纪念日”已经变成了一场每个人都能参与的、支持和团结性少数青年人的重要活动。



Tyler Clementi

(December 19, 1991 – September 22, 2010)



2019年10月17日是今年的紫色纪念日。在此，我们邀请所有的的法律人、媒体人和我们一起，重视欺凌问题，勇敢地参与到这一伟大的纪念日之中，支持性少数青少年，采取行动，反对欺凌！

# 如何参与紫色纪念日 #


1、将你的微信、微博、QQ等社交网站/软件的头像换上紫色图片（拉到文章下一部分，有可以加滤镜的二维码哦），或者在社交网站上分享自己身着紫色衣服或带有紫色元素的照片，并@同志平等权益促进会，或@中国彩虹媒体奖，大胆秀出自己发对欺凌的态度。

 

 

 

2、支持我们的工作，和我们一起努力让这个世界变得更好。每月捐出一杯咖啡的钱，就能让一个性少数青年免于自杀。
