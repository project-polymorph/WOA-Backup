深圳夕颜梧桐山登山活动

深圳夕颜
2017年02月28日 11:05

 各位客官，又一波户外活动来啦！！！夕颜敬请大家来参加！！！

 告别寒冷冬日，我们又迎来了深圳春意盎然、最适宜户外运动的好时节，辛勤工作、来回奔波的帅哥美女们是否早已按捺不住内心的骚动，想出来HAPPY一下呢。为了满足大家想出来游玩又不愿孤身一人行动的心愿，夕颜特意组织了新一波的免费户外活动——梧桐山漫步，诚邀各位一起攀登梧桐山、呼吸新鲜空气、拍摄满山美景。

 活动时间：2017.03.02（星期四）11:00—17:00

 活动地点：梧桐山/梧桐山艺术小镇

 集合时间：3月2日 上午11:00

 集合地点：

 1.夕颜工作室（请于10:30来此集合）或

 2.东门晒布地铁站D3出口附近麦当劳门口（去夕颜工作室不方便的可以11:00在此集合）


 温馨提示：

 请穿着运动鞋及宽松的衣服；

 请自备零食和饮用水（可在山下购买）；

 登山量力而行，切勿过度疲劳。
