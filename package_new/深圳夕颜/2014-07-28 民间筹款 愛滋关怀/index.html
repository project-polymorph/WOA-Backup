亲们所谓一方有难八方支援，当我们在遇到困难的时候都希望可以得到帮助，但是在别人遇到困难的时候我们是否也能伸出援助之手呢？我们现在有一位朋友，他不幸感染了艾滋病，而且得并发症（严重肺病）。现急需为他筹款治病，有心人请向深圳夕颜工作人员联系，多多益善少少无拘。善款可以汇到深圳市夕颜传播文化中心的账号9553 3072 0042 7970(建设银行)，捐款全数会用作这个朋友的医药费。谢谢！

家庭背景：

这位朋友他的母亲在他两岁时突然离开家庭，父亲曾带着他和哥四处去打探母亲的去向，最后也没找到。父亲也在他4岁时，和另一位女人再婚，重新建立一个家庭，并且有了两个孩子。因此也没有能力照顾他和哥哥。父亲放弃了养育他和哥哥，因此从小到大，都是由奶奶照顾，三人相依为命。可是，他奶奶在他21岁时（2008年）去世，而他哥哥也因为肠病，在他22岁时病逝。他突然失去两个在生命中最重要的人，整个世界都踏下来了，但他最后还是坚强地活着，靠着自己努力地工作，养活自己。他最亲的奶奶和哥哥去世后，他还有四个亲姑姑，特别是最小的姑姑，都疼他和照顾他。


经济困难原因：

他在2010年被确診出感染了後天免疫缺乏症候群（简称：艾滋病AIDS）在2014年6月，因为持续地發燒和咳嗽入院，在廣州的醫院接受治療，住院四天，花了兩万多，他把所有的积蓄和姑姑四处筹回来的钱都用在看病上，但因為負不起昂貴医疗费用，最后离开了医院，放弃治疗。

他从广州回到老家，因为已经没有经费治疗，身体越来越虚弱，后来透过身边的朋友四处找人帮忙，联系了有关的草根团体，暂时得到了大约$10000的救助，于是立刻送到了贵阳市的医院接受治疗。但因为他免疫力低，出现了不同的併发症，身体十分虚弱。医院给予了治疗，但

辛苦筹来的救助金很快在3天里花完。
