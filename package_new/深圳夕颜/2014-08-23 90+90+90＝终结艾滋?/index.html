消除艾滋病疫情，世界衞生组织提达到3个0的目标，即是零新增感染、零死亡和零歧视。为达到这三个伟大目标，联合国艾滋病规划署提出2020年逹致三个90%的目标，即：90%的感染者通过检测知道自己的感染状况，90%已经诊断的感染者接受抗病毒治疗；90%接受抗病毒治疗的感染者病毒得到抑制。但当局表示有困难！然而，这个意味着提出吃葯的策略，也未必受社群成员欢迎！

疾控专家：中国2020年实现艾滋“三个90%”防治目标有难度

(2014年08月15日中国新闻网)“联合国艾滋病规划署提出到2020年力争实现‘三个90%’的防治目标，就中国目前情况来看，达到这些目标有一定难度。”中国疾病预防控制中心性病艾滋病预防控制中心主任吴尊友15日在北京如是说。

　　当日，中国疾控中心在北京举办“艾滋病防治工作交流活动”。15位参加了第十二届世界艾滋病大会的代表和有关专家就这次会议报告的艾滋病最新进展等主题与媒体进行交流。

　　根据联合国艾滋病规划署2014年的报告，截至2013年底，全球现存活艾滋病病毒感染者和艾滋病病人共3500多万人，2013年全球新发感染210万人，艾滋病相关死亡150万人。联合国艾滋病规划署提出了“2030年终结艾滋病”的愿景，并提出“三个90%”防治目标，即：90%的感染者通过检测知道自己的感染状况，90%已经诊断的感染者接受抗病毒治疗；90%接受抗病毒治疗的感染者病毒得到抑制。

　　吴尊友表示，全球的艾滋病发现率平均在30%—50%之间，美国是艾滋病发现率最高的国家，已达到75%，中国目前是54%，要达到90%还是很难的。

　　“在中国43万存活的艾滋病病毒感染者和艾滋病病人中，有51%的人接受了抗病毒治疗。而在发达国家，治疗率也只能达到80%。”吴尊友说，在过去三年，中国有80%到86%的接受抗病毒治疗的感染者病毒得到抑制，实现“90%”相对比较容易。

　　吴尊友表示，以病死率为标准，中国艾滋病的治疗水平接近发达国家水平，儿童艾滋病治疗水平也和发达国家相近，但在基础研究、药品种类等方面还落后于发达国家。
