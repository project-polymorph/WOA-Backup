2009年2月，第一届MR. GAY大赛在加拿大成功举办。这是一项旨在让全世界的同志团结在一起，展现同志的内在与外在美的大赛。创办人希望以这个比赛为契机，让各个国家的同志都聚集在一起，分享经验，互相交流。


各个地区会先选出当地的MR.GAY，然后各个地区的冠军再参与角逐世界MR.GAY.

英国地区MR.GAY


今年，获得新西兰MR.GAY殊荣的是一位艾滋病感染者。他叫做Charlie Tredway。Tredway来自奥克兰，为新西兰艾滋病防治基金会(NewZealand Aids Foundation)工作，致力于宣传防治艾滋病。

Charlie Tredway

 

他赢得比赛之后非常激动，并表示，“因为有社区一直支持着我，我才能赢得这个比赛。”Tredway将在5月份代表新西兰参加在西班牙的举行MR.GAY世界总决赛。(就在今天开幕~）

 

 

虽然在新西兰，艾滋病确诊数量接近于零，但是艾滋病基金会认为，新西兰的艾滋病毒感染率正在上升。去年新西兰只有9例艾滋病病例报道，随着90年代抗逆转录病毒药物的问世，控制了艾滋病的发展。但是，这也使得公众对于艾滋病的警惕性降低了，大家渐渐觉得艾滋病离自己非常遥远。所以这次Tredway当选MR.GAY有着双重意义，不仅实现了大赛的初衷，还宣传了艾滋病防治。
