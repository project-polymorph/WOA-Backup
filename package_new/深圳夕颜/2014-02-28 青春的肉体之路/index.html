青春的肉体之路

何春蕤 深圳夕颜
2014年02月28日 17:24

 

在中文里，就和很多其他语言一样，总是以男人作为职业从业者的基本假设，若是女人来做，就要在前面加个「女」字，例如女总理、女老师、女工之类的。

 

性交易则是少数以女性为预设从业者的工作之一，若是男性来做，就会在前面加个「男」字，如男妓、男公关之类的。然而本书中的男性性工作者特殊之处，更在于他们并没有按照惯常的异性恋逻辑与女人交易，反而支撑起与男性互动交易的广大新市场。

 

正是因为这双重溢出常轨的「男」──与「男性」交易的「男性」性工作者──他们的生涯于是提供了一个窗口，让我们认识某些男性、某些男同性恋、某些男变女个体的社会位置、人生选择、以及苦辣酸甜，是如何标记了当代中国的经济转移与情欲革命。

 

经济起飞，总是意味着「机会」的出现，中国的崛起，在特殊的重点生产城市，例如深圳、东莞、上海等等，形成强大的吸引力。无数从未离开贫苦乡村或是梦想鸿鹄大志的年轻男性于是抓住了机会，流向城市。民工的孤立困苦和有限收入，相较于性工作的险恶与自主，提供了许多年轻男性崭新的体验，以他们的肉身迎向资本主义新形势的考验。

 

或许是因为没有贞节的传统压力，下海，在男性性工作者的叙述中似乎并没有常见的煎熬交战，相反的，言语之间偶而还透露着某种探险的动能，当然也有着认命的务实。即便对于自己的行业都有怕人发现的羞耻和恐惧，面对机会时，男性性工作者还是展现了灵活的动力。从离乡背井，远跑到香港、新加坡、马来西亚，漂泊的经验总是混杂着冒险的惊喜，在同文同语同乡同侪间打造着青春男儿的新生涯。

 

漂泊，是男性性工作者比女性同行更多见的感受。不过这种感受倒不一定来自身处异乡他国，也不一定来自性工作这个行业，反而更多的是来自个体的出身──他们的阶级，也来自个体所座落的巨变社会──中国。

 

正是因为快速变化，整个中国社会都在开发新的领域，各种生产事业已经首先蜕变，但是教育还没有赶上生涯规划的构想，那些成长环境不理想的个体当然看起来缺乏目标、缺乏专长。出于这样的背景，他们的眼界可能侷限于学习新兴服务业的小技术专业，例如调酒、看门、按摩等等，憧憬藉着有限的技能、藉着相互的引介，寻找更能满足人生需求的机会。作为背负着家庭重担和期望的儿子们，这些年轻男人必须在夹缝中创造出自信自尊的存活方式，才能继续遥想各式各样的未来。赚钱，是一个时而辛苦、时而慰藉的活动。

 

但是他们在性工作里却也同时认识了人生百态。他们奋力的摸索抗拒尔虞我诈的交易世界，不但要面对恶意剥削的老鸨，轻蔑霸道的警察和媒体，还有彼此竞争合作的同行；总是尽力在有限的互动中获取最大的利益，与客人亦交易、亦交友，有时情牵梦系却也各取所需。他们有些人在面对弱势消费者时表现了深刻的悲悯，也在遇到伪善傲慢的强势消费者时，用小技巧讨回一些公道。对他们而言，下海很难作为一种积极的人生选择，而总是带着随遇漂流的无奈。

 

作为血气方刚的男性性工作者，工作可以和情欲满足结合吗？答案是不一的。有些慢慢的在亲密互动中找到了感觉，有些完全区分金钱和情欲的对象，更有些一鱼两吃的左右逢源。在不甚理想的生活中，人们的个性和需求就主导了他们如何管控有限的能量和投注。令人好奇的是，谁又是那些购买这些服务的男人呢？他们的情欲状况又是怎样的呢？性工作者口中描绘的客户，好像总是隔着一层雾，看不轻面貌。

 

漂泊的男性性工作者都有着青春的肉体，这是他们几乎唯一的共通点。这些身体在汗水和体液的掩盖下，持续上演着人生最基本的戏码──存活。照着他们最敏锐的观察来看，随着经济的急剧变化，投入这行的男性身体也越来越多，越来越多样。在中国崛起的巨大身影下,男性性工作又将走出怎样精彩而卑微的人生呢？

何春蕤（台湾中央大学性／别研究室）
