由于求助朋友之前没有申请到任何的保障，所以得不到任何的补贴。草根团体曾努力地帮他去有联系有关新型农村合作医疗保险的政府部门，帮他申请到一些医疗补助，但是最后也不成功。


虽然他有姑姑，但因为姑姑也有自己的家庭，环境也不好，加上在广州的医院也资助了两万，所以姑姑也无能为力。

根据医院过去有关的个案参考，治疗的费用大约需要4-5万（根据病人的情况）。他现在还在医院里，但因为开始拖欠医疗费用。所以医院已经停药了。


他是一位坚强、努力、善良的人。他过去努力地照顾年老的奶奶和病重得的哥哥，虽然父母放弃了他，但他从没有放弃，靠自己的努力赚钱，给哥哥治病。他乐于助人，朋友有困难，都会尽力的帮忙。在工作上，也是勤奋，有责任感。病人是有权利得到治疗，希望大家可以伸出援助之手，帮他渡过这个难关。
