搭讪小哥哥是一门学问

原创 咖灰 爱同行 夕颜
2022年09月16日 19:55 广东

有些人看上去很优，外表是很大众的菜，可是不知道为什么，追求他的人却不多；有些人明明好像那么普通，身边跟他暧昧表白的人却没断过，以至于有时候会烦恼被好几个人同时追。



这是为什么呢？


有人说，是天菜的标准太高，太挑剔，所以产生了没有人追的假象。


不过，这只是问题的一方面。是一种可能性。同时，还有另一条重要的原因，与人互动的习惯。很多人把这叫做性格。性格也是影响人的受欢迎度的重要因素。但是这两个字会太抽象，让人不知道如何调整与改进。如果具体到说是跟人互动、创造美妙气氛的手法的话，那就是一门可以提升的技艺啦～


参加认识新朋友的聚会时，很多在群里线上聊天非常自信和多话的小伙伴，也可能陷入到无话可说的情景。


但是，人都出来了，再玩手机或者高冷沉默、放空呆滞都不是最好的选项。


无论是跟一群人相处，还是跟一位约会对象单独相处，如何调动相处气氛都是有学问的。


我不太推荐自high或者high爆全场的那种极为外向兴奋的手法，首先这很难学，另外，high爆的状态与讨你喜欢的小哥哥欢心完全是两码事。


有些擅长与人相处的人，凭借着中等的自身条件，却可以收服有好感的人。这要怎么做到，才是敲黑板的重点。 

01

# 自信 #


自信是知道你拥有哪些筹码。并且对搭讪、主动交流能产生的后果有了解，不害怕被拒绝或发生预料之外的效果。



多与人互动，能够提升你的自信。而且保持每周至少一次的社交习惯，能够让你不那么在乎每一次的搭讪、聊天、创造交流气氛，不会因为遇见了一个喜欢的类型的人，就患得患失。


搭讪、约会及相处，就像是下棋游戏，胜败乃兵家常事。有一定量的运筹帷幄的经验，能让你更加自信。


02

# 读懂他人 #


在一群人的聚会中，观察不同人是否有什么心理需求，能让你在适当的情况说出合适的话，创造出一些欢笑、正面的气氛效果。 


在跟一个人的约会中，体会对方的心情、正在关注或沉醉于什么内心状态，能让你更接得出对方的出招。有些人跟身边人独处时，说话有半句没半句，如果你跟他的状态不在近似的轨道上，可能会愣住，没头没脑不知道对方想表达什么。


而如果你拼命努力跟随别人的思维轨道，又会很容易累，而且显得没有自我，没有幽默感和自信。


最好的读懂他人的状态是，既轻松保持令自己舒适的状态，不关闭独处时那种自得其乐的自我状态，同时也向他人打开一扇窗，用五官和直觉的试探去了解别人关注什么，想要分享什么，想要避免什么。


当别人对你放下防御，拥有正面感觉甚至期待你的出招时，你就可以采取更多想要的下一步举动了。


03

# 乐在当下 #


所有的快乐都是当下发生的，所有的约会实际上也都不会有下次。因为下次即使是跟同样的人在一起，发生的事情和产生的效果也全然不同了。 


所以，不要纠结于分析，而是采取大体合适的方式 ，自由发挥，轻松的笑话，随意的触碰，不必澄清辩解别人对自己的什么误会，不必吝啬计较眼前的人们是否值得你热心。


只要把每一次与人相处，都当作漫长的工作睡觉独处之外的一次难得机会，发现眼前人的可爱之处，忽略或包容彼此的不同，就可以创造出许多快乐的接触了。 


当你拥有了一定的经验，不需要改变自己的性格，只要像打毛衣那样（越打越熟了），放松地享受与人相处的时间，你就可以搭讪许多有好感的人，认识和发展许多值得来往的朋友。


多一点丰富多彩的生活，试探舒适圈以外的新的快乐。 


多一个快乐的打招呼，加几句轻松往来的闲聊。


无论是为你爬楼辛苦送饭的骑手小哥，还是每天在写字楼容易遇到的不知名但印象不错的人，以及在同一个群组，聚会可能见到但很少交流的网友……


所有的人，都在你的6层扩展脉络的某一层。


如果你对他有兴趣，就不要闷着不行动，也不要采取破坏自己心态平衡的激进行动。



多一点眼神的对视，适时的试探和不经意的碰触。


多一点把自己当作别人的礼物，肯定自己的同时，也给予别人选择的自由。


只要每次轻松共度多一秒，一分钟，不害怕，不留恋，你的朋友会更多，你的美妙奇遇，也会在该来的时候陆续到来。
