采访向阳医生：分享HIV治疗工作中的心得与经验

原创 咖灰 爱同行 夕颜
2023年05月23日 14:05 广东


    在《听说》发布会上，我们请到了深圳市罗湖区人民医院感染性疾病科副主任向阳医生作为我们的主讲嘉宾，今天，更是有幸找到向主任接受采访。向阳医生在HIV感染者治疗工作中拥有丰富的经验和心得，同时还参与了罗湖区爱心门诊的创建。下面，是关于他在工作中对HIV感染者群体的看法。 

  ——向阳医生采访问答—— 



向阳医生，能否分享一下您对HIV感染者群体的印象和理解？

HIV感染者群体是一个非常特殊的人群，他们面临着巨大的压力和歧视。作为医生，我们要尽可能地为他们提供全面、专业的医疗服务，帮助他们尽早得到诊断和治疗，恢复健康。


您如何看待当前社会对HIV感染者群体的接纳和包容度？

我认为，当前社会对HIV感染者群体的接纳和包容度仍有待提高。很多人对HIV感染者存在误解和偏见，我们需要通过教育和宣传，让更多的人了解HIV的真相，减少歧视和偏见。


我们有收到小伙伴的一些疑惑，也想跟向主任了解一下：有小伙伴说，担心去其他医院就诊时，医生会看到他们患有其他疾病的记录，因为之前使用过医保购买药品。您认为这种担忧有道理吗？

这种担忧是没有必要的。医生看不到患者在其他医院开过的药品记录。关于患者的基础病情，需要患者主动告知医生。以前在急诊时，患者需要插入社保卡，而现在这一过程已取消，患者可以直接在网上使用社保卡号付费。


您在文章中提到了与爱同行志愿者的合作，能否详细说说？

爱同行志愿者是一个非常优秀的团队，他们在我们的工作中发挥了重要作用，例如帮助HIV感染者寻找医疗资源、关注患者的生活和心理状况等。记得有一次，我们在发热门诊中遇到了一个新冠疑似病例，后被确诊为HIV感染。但是他是属于三无人员，没有亲属或工作，也没有人照顾。我们医院遇到“三无”的病人，要找民警协助，但因为手续过于繁杂，我帮他去社区派出所开无犯罪证明等，跑了很多趟，始终也还是办不下来。


这件事就是您在《听说》小册子中提及跟轩轩合作帮助病人的那次经历吧？

对，这个人自从病了以后工作也没了，那时候是疫情期间，他发热到我们发热门诊去看，做了CT，就是肺部有感染，就作为疑似病例，刚好他排除新冠可以出院那天是我值班，所以我就帮他办出院手续。但检测后来发现他HIV初筛阳性，以及有肺孢子菌肺炎，鹅口疮，真菌感染，有段时间讲话都讲不了，需要转院。我看他走路困难，他又一个人住在出租屋，我就有点不放心，怕他在家里出事。我就第二天去找他，第二天应该是礼拜一，把他接到爱心门诊就抽了一个确证，然后把他送到三院去，想到他要陪护，我就跟爱同行的雪明、轩轩讲了这事，轩轩就过来帮忙了。


那你们自己也有工作要忙，如何顾得过来每天照顾他呢？

确实，我们担心这个病人没有自理生活能力，但又没办法整天陪护他，那时候是快过年了，轩轩他们也非常忙，所以我和轩轩共同商议，出钱为他请了护工陪护。病人住院一住三个月很快就没钱了，我们帮他先后就交了好多次。当时也想着我们可以在这个经费里面走，结果也是无奈走不下来，当时情况挺急的，走流程都不知道怎么走。这个病人前后可能花了有4万多元，主要就是住院费，轩轩自己出了1万多，就是还包括那些陪护、餐费等。反正后来报不了就不报了，我们就自己承担了。


其他病人还发生过有类似情况吗？

其他的病人，依靠我们帮忙的也有。无论是帮HIV感染者办各种手续证明，还是关注他们的生活和心理状况，甚至用自己的钱来垫付他们的医疗费用，我们都做过不少。这些工作都是为了改善患者的处境，每一位病人都有他们自己的难处。


在您的工作中，HIV相关的占了多少比重？您对这个领域的工作有什么感想？

HIV方面的，占了我大部分工作内容。我们要为HIV患者提供综合的医疗服务，包括病情诊断、治疗方案的制定，还有后续的跟进和关怀。有时候，这些工作不仅仅局限于医院，还会涉及到更广泛的社区服务，关心他们的个体情况。作为医生，应该尽量去帮助病人，不仅是为了治疗他们的疾病，也更是为了给他们带来希望和信心。无论是HIV感染者，还是其他任何病患，他们都需要得到尊重和关怀。


说到偏见，我们知道U=U是一个经常提及的话题，您对U=U理念的看法以及它对患者的重要性。

U=U理念非常重要，即病毒载量持续低于检测下限的患者不会通过性途径传播病毒。这一理念对患者至关重要，因为它有助于他们重新融入社会，消除歧视和污名化。同时，它也能鼓励更多人进行检测和治疗，提高全社会对HIV的认知和关注度。


非常感谢您的分享。接下来，请问您对于HIV感染者群体和心理健康有哪些建议？例如，如何应对歧视和压力等问题？

我建议我们尽可能地帮助HIV感染者恢复自信和自尊。在治疗过程中，除了提供医疗支持，我们还需帮助他们解决社会和心理方面的问题。面对歧视和压力时，我们要告诉他们，HIV感染并非他们的错，需要以平等、包容的态度对待每个人。同时，我们还应提高公众对HIV感染者的认识和了解，让更多人认识到他们同样拥有权利和尊严。


    在这次对话中，我们感受到向阳医生对HIV感染者的深深理解和无私的奉献。他的分享也让我们更深入地了解到了HIV感染者群体所面临的挑战，以及他们所需要的支持。向阳医生的专业知识和热忱，让我们深受启发和从中获得力量。


    期待未来，有更多的机会和医生们进行交流，我们会将他们的故事、知识和心得分享给大家。
